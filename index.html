Gukere Team

Gukere es una iniciativa empresarial que tiene como objetivo crear aplicaciónes web capaces de utilizarse en cualquier tipo de dispositivos (PC, móvil, tablet, TV, ...). Para ello se cree en el uso de estándares creados por el W3C, consorcio gestor de las bases de desarrollo de sitios web.

Tecnología

En los desarrollos, Gukere hace uso de HTLM5, la última revisión del lenguaje básico de la World Wide Web, HTML. A pesar de que todavía se encuentra en modo experimental, HTML5 ofrece mútiples avances, mejoras y ventajas en los desarrollos. A diferencia de "web 2.0", esta vez HTML5 tiene un norte definido, un grupo de empresas y fundaciones apoyando y una visión.

HTML5 es un lenguaje de modelado de información por etiquetas. CSS3 es un lenguaje de estilos en cascada. Javascript es un lenguaje de programación. Y todos, todos, se engloban alrededor del concepto HTML5.

Servicios

Los servicios principales que ofrece Gukere son desarrollos web universales a medida. ¿Universales por qué? Porque están pensados para que se ejecuten de manera correcta en cualquier tipo de dipositivo. Para ello Gukere hace uso del patrón Responsive Design y la tecnología HTML5.

Muy bien, ¿Y esto qué supone?

Desde los últimos años, los desarrollos para dispositivos móviles se han realizado dependiendo del sistema operativo. Es decir, se han realizado un desarrollo específico para iOs, Android, Blackberry, etc. Con el patrón Responsive Design y HTML5 se desarrolla una aplicación web capaz de ejecutarse en todos los dispositivos y aprobechar, practicamente, todas las funcionalidades.

El diseño web adaptable (Responsive Design) es una técnica de diseño y desarrollo web que mediante el uso de estructuras e imágenes fluidas, así como de media-queries en la hoja de estilo CSS, consigue adaptar el sitio web al entorno del usuario.
